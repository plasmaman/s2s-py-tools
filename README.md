# s2s-py-tools (working title)

Procedural Python code for reading ECMWF S2S forecast data and generating forecast products.

## Status
Initial working version:
- ECMWF native forecast reader (cf/pf)
- Daily precipitation computation
- Grid loading
- Hindcast date matching
- Bias correction

This repository starts from a clean procedural rewrite of earlier class-based code.

A validation paper for the dry-spell forecast is publshed in Climate Dynamics: https://doi.org/10.1007/s00382-025-07920-4
